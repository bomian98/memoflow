## English

Thank you for your interest in this project. I truly appreciate everyone who wants to help improve it.

However, this project currently **does not accept Pull Requests**. If you open a PR, it may be closed without being merged. This is not because your contribution is not valuable, but because of how this project is maintained.

This project is primarily developed with the help of AI. As the maintainer, I do not currently have the ability or capacity to fully review external code contributions for correctness, usability, security, stability, and consistency with the existing implementation.

To keep the development process controlled and maintainable, this project does not accept code contributions through Pull Requests at this time.

Feedback is still welcome. You can help by opening Issues to report bugs, suggesting improvements, sharing use cases, or providing clear reproduction steps.

Thank you for your understanding and support.

## 中文

感谢你对本项目的关注，也非常感谢所有愿意帮助改进项目的人。

不过，目前本项目 **不接受 Pull Request**。如果你提交了 PR，它可能会被直接关闭。这并不是因为你的贡献不重要，而是出于项目维护方式和质量控制的考虑。

本项目的代码主要由 AI 辅助开发完成。作为项目维护者，我目前没有足够的能力或精力去完整审核外部提交代码的可用性、安全性、稳定性以及与现有实现的一致性。

为了保证项目开发过程的可控性，避免引入我无法充分验证的问题，本项目暂时不接受任何形式的代码 PR。

不过，反馈仍然非常欢迎。你可以通过提交 Issue 反馈 Bug、提出功能建议、分享使用场景，或提供清晰的问题复现步骤来帮助项目改进。

感谢你的理解和支持。
